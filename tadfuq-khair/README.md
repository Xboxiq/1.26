# تدفّق الخير — منصة خدمات المشتركين

منصة واجهة أمامية لمركز خدمات مشتركي الكهرباء (RASAFA · CS HUB).  
**٣١ خدمة** عبر **٤ أقسام**: اشتراكات · فنية · فواتير · شكاوى.

> هذا الريبو **جديد ومستقل** — الريبو القديم [`Xboxiq/1.26`](https://github.com/Xboxiq/1.26) يبقى كما هو دون حذف.

---

## البدء السريع

افتح `portal.html` في المتصفح — بوابة الدخول لكل أجزاء المشروع.

| الملف | الوصف |
|:--|:--|
| [`portal.html`](portal.html) | **بوابة الدخول** — ابدأ من هنا |
| [`final3d.html`](final3d.html) | المنصة المعتمدة (3D tilt) |
| [`final.html`](final.html) | نفس المنصة بدون 3D |
| [`اتجاهات التصميم.html`](اتجاهات%20التصميم.html) | **جديد** — ١٠ اتجاهات بصرية للاختيار |
| [`كل المقترحات.html`](كل%20المقترحات.html) | أرشيف المختبرات والتصاميم |
| [`compare.html`](compare.html) | مقارنة جنباً إلى جنب |
| [`DESIGN_SYSTEM.md`](DESIGN_SYSTEM.md) | مرجع نظام التصميم الكامل |
| [`CLAUDE.md`](CLAUDE.md) | قواعد التطوير للمساعدين |

## المختبرات

| المختبر | المحتوى |
|:--|:--|
| اتجاهات التصميم | ١٠ هويات بصرية + ٤ تركيبات مقترحة |
| بنك الاعتماد | أفكار قابلة للتطبيق على العناصر الحقيقية |
| الأيقونات والبطاقات | ١١ موجة تصميم |
| الأزرار والطقوس | أنماط CTA والتفاعلات |
| مسرح المفاهيم | ٥ مشاهد من عالم الكهرباء |
| نظام التصميم | الدليل البصري الحي |

**الترشيح:** انقر ☆ في أي مختبر — القائمة تُحفظ في `localStorage` ويمكن نسخها.

---

## البنية

```text
data.js              ← مصدر البيانات (خدمات، أجور، حالات)
final*.jsx / .css    ← المنصة المعتمدة (طبقات CSS)
ds/                  ← توكنز نظام التصميم
v2/                  ← نسخة ديوان تجريبية
versions/v1-approved ← snapshot محمي — لا يُعدَّل
```

**إضافة خدمة:** سطر واحد في `data.js` — الواجهة تشتق كل شيء تلقائياً.

## التشغيل المحلي

```bash
# Python
python3 -m http.server 8080

# أو Node
npx serve .
```

ثم افتح: `http://localhost:8080/portal.html`
