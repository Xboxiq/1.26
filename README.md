# تدفّق الخير — منصة خدمات المشتركين

منصة واجهة أمامية لمركز خدمات مشتركي الكهرباء. **٣١ خدمة** · **٤ أقسام**.

> الملف الأصلي `تدفق الخير نظام متكامل (3).zip` **محفوظ** في هذا الريبو كنسخة احتياطية.

---

## ⚠️ مهم — كيف تشغّل المنصة

الملفات **لا تعمل** بفتح HTML مباشرة من المستكشف (`file://`).  
يجب تشغيل خادم محلي:

```bash
./start.sh
# أو
python3 -m http.server 8080
```

ثم افتح في المتصفح:

| الرابط | الوصف |
|:--|:--|
| http://localhost:8080/portal.html | **ابدأ من هنا** |
| http://localhost:8080/final3d.html | المنصة المعتمدة |
| http://localhost:8080/اتجاهات%20التصميم.html | اختيار الهوية البصرية |

---

## هيكل المشروع

```text
portal.html          ← بوابة الدخول
final3d.html         ← المنصة المعتمدة
data.js              ← بيانات الخدمات (مصدر واحد)
final*.jsx / .css    ← مكوّنات المنصة
ds/                  ← نظام التصميم
v2/                  ← نسخة ديوان تجريبية
versions/v1-approved ← snapshot محمي
```

## استكشاف الأخطاء

| المشكلة | الحل |
|:--|:--|
| شاشة بيضاء | شغّل `./start.sh` ولا تفتح الملف مباشرة |
| لا يظهر React | تحقق من اتصال الإنترنت (CDN) |
| ZIP فقط | الملفات مفكوكة في جذر الريبو — لا حاجة لفك يدوي |
