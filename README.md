# تدفّق الخير — مشروع المنصة

## الريبو القديم (محفوظ)
- الملف الأصلي: `تدفق الخير نظام متكامل (3).zip` — **لم يُحذف**
- الفرع `main` يبقى كما هو

## النسخة الجديدة الكاملة
المشروع الكامل مع كل الإضافات في المجلد:

**[`tadfuq-khair/`](tadfuq-khair/)**

ابدأ من: [`tadfuq-khair/portal.html`](tadfuq-khair/portal.html)

### الإضافات الجديدة
- `portal.html` — بوابة دخول لكل أجزاء المشروع
- `اتجاهات التصميم.html` — ١٠ اتجاهات بصرية للاختيار
- زر نسخ في المختبرات (lab_tools)
- مقارنة محدّثة (final3d / v2 / vercel)
- `README.md` وتوثيق محدّث

### تشغيل محلي
```bash
cd tadfuq-khair
python3 -m http.server 8080
# افتح http://localhost:8080/portal.html
```

### ريبو مستقل (اختياري)
لإنشاء ريبو GitHub منفصل `Xboxiq/tadfuq-khair`:
```bash
cd tadfuq-khair
git init -b main
git remote add origin https://github.com/Xboxiq/tadfuq-khair.git
git push -u origin main
```
