# تدفّق الخير — منصة خدمات المشتركين

## افتح المنصة الآن

| | الرابط |
|:--|:--|
| **المنصة الرئيسية المعتمدة** | **https://xboxiq.github.io/1.26/final3d.html** |

> المسار المعتمد للعرض الآن هو `final3d.html` فقط.
> أي دخول عبر `index.html` أو `portal.html` يتم تحويله تلقائياً إلى المنصة الرئيسية.

---

منصة واجهة أمامية لمركز خدمات مشتركي الكهرباء. **٣١ خدمة** · **٤ أقسام**.

> الملف الأصلي `تدفق الخير نظام متكامل (3).zip` **محفوظ** في هذا الريبو كنسخة احتياطية.

---

## تشغيل محلي (اختياري)

```bash
./start.sh
# أو: python3 -m http.server 8080
```

| الرابط | الوصف |
|:--|:--|
| http://localhost:8080/final3d.html | المنصة الرئيسية المعتمدة |

---

## هيكل المشروع

```text
index.html / portal.html ← تحويل تلقائي إلى المنصة الرئيسية
final3d.html         ← المنصة الرئيسية المعتمدة
data.js              ← بيانات الخدمات (مصدر واحد)
final*.jsx / .css    ← مكوّنات المنصة
ds/                  ← نظام التصميم
versions/v1-approved ← snapshot محفوظ للمراجعة فقط، غير معروض كمسار رئيسي
```

## ملاحظة العرض

المختبرات والنسخ التجريبية محفوظة داخل الريبو للمراجعة فقط، لكنها ليست جزءاً من مسار العرض المعتمد.
المسار الوحيد الذي يُعرض للمستخدم هو `final3d.html`.
