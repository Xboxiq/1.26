تدفّق الخير

نظام التصميم الرسمي لمنصة خدمات المشتركين. كل ما يلزم لإضافة صفحة أو خدمة جديدة بنفس الروح: الألوان، الخطوط، المواد الزجاجية، الحركة، والأيقونات المرسومة خصيصاً.

01 الألوان

قماش دافئ هادئ (روح Notion) + أربعة ألوان أقسام مهدّأة لا تصرخ أبداً. القاعدة: لون القسم يُستعمل بسُلّم شفافية ثابت — 11% تعبئة، 22% حدود، 100% نص/أيقونة فقط.

03 الأيقونات المرسومة

مجموعة مرسومة خصيصاً للمنصة — 24×24، خط 1.75، أطراف دائرية. لغة كهربائية صريحة: مقياس، عمود، كابل، برج. تُستعمل داخل بلاطة squircle بلون القسم. Material Symbols تبقى للأفعال العامة فقط (بحث، إغلاق، طباعة).

04 المواد والأسطح

مصدر ضوء واحد من الأعلى: كل سطح مرفوع يحمل خط specular علوي. ثلاث مواد: ورق (بطاقات)، زجاج مصنفر (شرائط وحلقات)، وقطعة مصنّعة (أزرار).

ورق — البطاقاتborder 0.5px + specular + shadow-xs
زجاج — Spotlight والشرائطblur(34px) saturate(1.9)
غسلة القسم — البانل المفتوح9% → 4% → 0 @155deg

06 الحركة

كل الحركات بـ spring واحد: cubic-bezier(0.34, 1.3, 0.45, 1). مايكرو 180ms، انتقالات 420ms، تمدد البانلات 620ms. تحترم prefers-reduced-motion دائماً.

--f-spring · 620ms

07 إضافة خدمة جديدة

الخدمات بيانات لا واجهات: أضف سطراً واحداً في data.js وستظهر تلقائياً في كل مكان — الأقسام، المحفظة، Spotlight، صفحة الخدمات، وحساب الأجور.

// data.js → window.SERVICES { code: 'CS0012', // القسم + رقم تسلسلي section: 'CS', // CS | CT | CB | CA name: 'اسم الخدمة', sla: 3, // أيام الإنجاز popularity: 50, // 0-100 (ترتيب Spotlight) fixedPrice: 25000, // أو hasPrice + priceNote icon: 'tdq-meter', // من ds_icons.js أو Material fees: [ // اختياري: بنود الأجور للفورمة { label: 'أجور الكشف', amount: 15000 }, ], }
